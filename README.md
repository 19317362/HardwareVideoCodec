##HardwareVideoCodec
HardwareVideoCodec is an media encode library for Android.

##Features
1.Support hard & soft encode.
2.Record video & audio. Pack mp4 through MediaMuxer.
3.Use OpenGL to render and support special effects.
4.More Features
